# paper-format-automation

A local-first automation toolkit / Codex skill for Chinese journal manuscript template alignment.

## v1 scope
- journal template rule extraction from `.docx`
- manuscript/template diff report generation
- conservative auto-formatting for deterministic items
- `.doc` template preprocessing through Word COM on Windows

## Main entry points
- `SKILL.md`
- `scripts/extract_template_rules.py`
- `scripts/check_manuscript.py`
- `scripts/format_manuscript.py`
- `scripts/run_pipeline.py`

## Notes
This repository is the first open-source cut of the local skill used to align Chinese journal manuscripts against official Word templates.
